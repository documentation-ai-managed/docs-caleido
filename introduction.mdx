## Overview

Caleido provides real-time cash forecasting by connecting your accounting data from platforms like QuickBooks or Xero. You gain accurate predictions as business conditions change, helping you spot risks early and plan investments confidently. This SaaS platform simplifies cash flow management for finance teams.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create your account at `https://app.caleido.io/register`.
  </Step>
  <Step title="Connect Data" icon="link">
    Link your accounting provider via OAuth.
  </Step>
  <Step title="View Forecast" icon="trending-up">
    Access your first cash forecast dashboard.
  </Step>
</Steps>

<CodeGroup tabs="API Integration">
  ```javascript
  // Fetch latest forecast
  const response = await fetch('https://api.example.com/v1/forecasts', {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  });
  const data = await response.json();
  console.log(data.cashProjection);
  ```
  ```python
  # Fetch latest forecast
  import requests
  response = requests.get(
      'https://api.example.com/v1/forecasts',
      headers={'Authorization': 'Bearer YOUR_API_KEY'}
  )
  data = response.json()
  print(data['cashProjection'])
  ```
</CodeGroup>
