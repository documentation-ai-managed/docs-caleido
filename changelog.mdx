## Upgrading to the Latest Version

Follow these steps to ensure a smooth upgrade.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Export your forecasts and settings.

````bash
curl -X GET https://api.example.com/v1/forecasts/export \
  -H "Authorization: Bearer YOUR_TOKEN"
````

  </Step>
  <Step title="Update Client SDK" icon="download">

    <Tabs>
      <Tab title="npm" icon="package">
````bash
npm update caleido-sdk@latest
````
      </Tab>
      <Tab title="Python" icon="code">
````bash
pip install caleido-sdk --upgrade
````
      </Tab>
    </Tabs>

  </Step>
  <Step title="Migrate Auth" icon="shield">
    Update to OAuth 2.0 flows. See the [authentication guide](/authentication) for details.
  </Step>
  <Step title="Test Forecasts" icon="zap">
    Run a sample forecast to verify everything works.
  </Step>
</Steps>

<Callout kind="tip">
  For v1.0.0 users, review breaking changes in API endpoints. Use the new `{https://api.example.com/v1}` base path.
</Callout>
